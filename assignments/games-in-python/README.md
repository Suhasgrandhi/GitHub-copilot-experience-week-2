
# 📘 Assignment: 🎮 Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input.

## 📝 Tasks

### 🛠️	Create the Hangman Game

#### Description
Implement a Hangman game where players guess letters to reveal a hidden word before running out of attempts. The program should provide feedback on progress and track remaining guesses.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (_ _ _ format)
- Track incorrect guesses remaining
- End when word is guessed or attempts exhausted
- Display clear win or lose messages
